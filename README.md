# flask-api-sqlalchemy

## Installation

```bash
virtualenv flask_venv -p /usr/bin/python3
source flask_venv/bin/activate
pip install -r requirements.txt

# start database
python3 build_database.py

```

### Start server
`python3 server.py`

### Stop virtual environment

`deactivate`
